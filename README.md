# 🐻 Interactive Bear Login - Flutter & Rive

An interactive and dynamic login interface built with **Flutter** that features a reactive animated bear. The character responds in real-time to user interactions, creating a unique and engaging user experience. 🚀

---

## 🌟 Features

* **Dynamic Eye Tracking:** The bear follows your cursor/typing progress while entering your email ("Chisme" mode).
* **Privacy Mode:** The bear covers its eyes when you click on the password field to ensure your privacy.
* **Interactive Feedback:** Integrated state machine triggers for success and failure animations during the login process.
* **Responsive Design:** A fluid layout that adapts to different screen sizes using `MediaQuery`.
* **Modern UI:** Clean interface with rounded borders and intuitive icons to toggle password visibility.

---

## 🎨 What is Rive & State Machine?

### **Rive** 🎬
Rive is a real-time interactive design and animation tool. Unlike traditional GIFs or videos, Rive animations are vector-based, lightweight, and can be manipulated via code in real-time.

### **State Machine** ⚙️
The State Machine is the **"brain"** behind the animation. It defines the logic and transitions between different visual states (e.g., from "idle" to "hands up") based on inputs received from the Flutter application.
